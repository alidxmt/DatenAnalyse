# Data Analysis Session: Key Takeaways

* **Markdown & Environment**
    * **Setup:** VS Code extensions & Live Preview (`Ctrl+Shift+V`)
    * **Syntax:** Headings, Tables, Checklists, and Code Blocks

* **Data Analysis Strategy**
    * **Workflow:** Problem → Collection → Preparation → Analysis → Storytelling
    * **DIKW:** Data → Information → Knowledge → Wisdom
    * **Categories:** Descriptive, Diagnostic, Predictive, Prescriptive

* **Goals & KPIs**
    * **SMART:** Specific, Measurable, Achievable, Relevant, Time-bound
    * **Metrics:** Actionable KPIs vs. Vanity Metrics

* **Python & Logic**
    * **Control Flow:** `for` loops, `range()`, and `if/elif/else`
    * **Structure:** Reusable `def` functions
    * **Method:** "Reverse Practice" (commenting existing code)

* **Data Exploration (Pandas)**
    * **Inspection:** `.info()`, `.describe()`, and `.head()`
    * **Cleaning:** Handling missing values and duplicates
    * **Practice:** Analyzing food composition and snack datasets
