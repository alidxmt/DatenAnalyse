# 🖥 PART 1 — Install a Markdown Extension

## Step 1: Open VS Code

Start Visual Studio Code on your computer.

---

## Step 2: Open Extensions Panel

You can open Extensions in two ways:

- Click the **Extensions icon** on the left sidebar  
OR  
- Use shortcut:

  - `Ctrl + Shift + X` (Windows/Linux)  
  - `Cmd + Shift + X` (Mac)

---

## Step 3: Search for Markdown Extension

In the search bar, type:

```
Markdown Preview
```

Recommended extensions (install ONE):

- Markdown Preview Enhanced  
- Markdown All in One  

Click **Install**.

---

# 📂 PART 2 — Create Your Markdown File

## Step 4: Create New File

1. Click **File → New File**
2. Save it as:


Make sure the file ends with `.md`

## 📚 What is Markdown?

Markdown is a lightweight markup language used to format text.

It allows you to create:

- Headings
- Lists
- Tables
- Code blocks
- Links
- Images
- Quotes

# 👀 PART 4 — Open Markdown Preview

To preview your Markdown:

### Option 1 — Keyboard Shortcut

- `Ctrl + Shift + V` (Windows/Linux)
- `Cmd + Shift + V` (Mac)

### Option 2 — Right Click

- Right-click inside the file
- Select **Open Preview**

You should now see formatted content on the side.
