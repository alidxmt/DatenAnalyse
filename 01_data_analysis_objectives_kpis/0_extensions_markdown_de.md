# 🖥 TEIL 1 — Markdown-Erweiterung installieren

## Schritt 1: VS Code öffnen

Starte Visual Studio Code auf deinem Computer.

---

## Schritt 2: Erweiterungen öffnen

Du kannst die Erweiterungen auf zwei Arten öffnen:

- Klicke links in der Seitenleiste auf das **Erweiterungen-Symbol**  
ODER  
- Verwende die Tastenkombination:

  - `Strg + Umschalt + X` (Windows/Linux)  
  - `Cmd + Umschalt + X` (Mac)

---

## Schritt 3: Nach einer Markdown-Erweiterung suchen

Gib im Suchfeld ein:

```
Markdown Preview
```

Empfohlene Erweiterungen (installiere EINE davon):

- Markdown Preview Enhanced  
- Markdown All in One  

Klicke auf **Installieren**.

---

# 📂 TEIL 2 — Markdown-Datei erstellen

## Schritt 4: Neue Datei erstellen

1. Klicke auf **Datei → Neue Datei**
2. Speichere die Datei unter folgendem Namen:

```
markdown-uebung.md
```

Wichtig: Die Datei muss auf `.md` enden.

---

# ✍️ TEIL 3 — Markdown-Inhalt schreiben

Kopiere den folgenden Inhalt in deine Datei.

---

## Beispielinhalt

## 📚 Was ist Markdown?

Markdown ist eine einfache Auszeichnungssprache zur Formatierung von Text.

Man kann damit erstellen:

- Überschriften
- Listen
- Tabellen
- Codeblöcke
- Links
- Bilder
- Zitate

---

## 🔠 Textformatierung

**Fetter Text**

*Kursiver Text*

***Fett und kursiv***

---

## 📋 Listen

### Ungeordnete Liste
- Apfel
- Banane
- Orange

### Geordnete Liste
1. Erster Schritt
2. Zweiter Schritt
3. Dritter Schritt

## 💻 Code-Beispiel

Inline-Code: `print("Hallo")`

Codeblock:

print("Hallo Markdown!")
for i in range(3):

# 👀 TEIL 4 — Markdown-Vorschau öffnen

Um die Vorschau zu sehen:

### Option 1 — Tastenkombination

- `Strg + Umschalt + V` (Windows/Linux)
- `Cmd + Umschalt + V` (Mac)

### Option 2 — Rechtsklick

- Rechtsklick in die Datei
- „Vorschau öffnen“ auswählen

Jetzt solltest du den formatierten Text sehen.
